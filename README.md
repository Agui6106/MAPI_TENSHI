
# MAPI_TENSHI Version alpha 0.1
Desarrollo de robot con visión robótica embebido implementado Raspberry y ESP32


# Funciones.
    * Recepecion de sensores en una aplicacion principal
    * Envio de un stream de una camara web remota
    * Vision robotica basica (Deteccion de colores y contornos)
    * Toma de fotografia en caso que se neceiste
    * Comunicacion MQTT entre Raspberry Pi y ESP32


# Bugs conocidos.
Saturacion del canal MQTT cuando se envian datos numericos de la aplicacion al ESP32
